  - Change 1: `scripts/sync-milestones.sh` (~165 lines) — idempotent `ROADMAP.md` → GitHub Milestones sync. Parses `v<MAJOR>.<MINOR>[<letter>]: <Title>` entries via awk (same regex family as `check_spec_taxonomy.sh`), fetches existing Milestones via `gh api`, plans the diff (CREATE / UPDATE / SKIP / LOCKED), optionally applies with `--apply`. **Dry-run by default** — tools that mutate shared state require explicit opt-in. **LOCKED on closed** — closed Milestones back released tags; never mutated. `--repo` + `--file` overrides for testing.
